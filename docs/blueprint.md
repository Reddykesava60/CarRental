# **App Name**: Car Rental Simplified

## Core Features:

- Car Browsing: Allow users to view all available cars with details such as name, model, and price per hour/day.
- Car Booking: Enable users to book a car directly by providing their name and desired rental duration.
- Return Car: Implement functionality for users to return a rented car using their booking ID.
- Add New Car: Allow administrators to add new cars to the system with details like name, model, and price.
- Update Car Details: Enable administrators to update car details such as price and model.
- Toggle Availability: Provide a feature for administrators to toggle car availability (active/inactive).
- View Bookings: Allow administrators to view all bookings in the system and mark a car as returned.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and reliability, essential for a rental service.
- Background color: Light gray (#F5F5F5), offering a neutral backdrop that ensures readability and focuses attention on content.
- Accent color: Orange (#E65100) to highlight interactive elements and important actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, and 'Inter' (sans-serif) for body text to ensure readability.
- Use clean, modern icons to represent different car types and actions.
- Design a responsive layout that adapts to different screen sizes.
- Incorporate subtle animations to provide feedback on user interactions.
